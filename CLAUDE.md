# base-web
## Propósito
Este proyecto es una plantilla base clonable para proyectos de clientes.
NO es un proyecto final. Cada cliente tendrá su propio fork/clon.

## Al trabajar aquí, Claude debe:
- Mantener el código genérico y reutilizable
- Evitar hardcodear datos específicos de ningún cliente
- Pensar en cada decisión como si afectara a N proyectos futuros
- Preferir configuración por variables de entorno o archivos .env
